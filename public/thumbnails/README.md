# Sign Language Video Thumbnails

This directory should contain the following thumbnail images for the sign language videos:

## Required Images:
- `sign-language-greetings.jpg` - Basic greetings video thumbnail
- `sign-language-numbers.jpg` - Numbers 1-20 video thumbnail
- `sign-language-family.jpg` - Family members signs thumbnail
- `sign-language-colors.jpg` - Colors in sign language thumbnail
- `sign-language-phrases.jpg` - Common phrases thumbnail
- `sign-language-weather.jpg` - Weather and seasons thumbnail

## Image Specifications:
- **Format:** JPG or PNG
- **Size:** 640x360px (16:9 aspect ratio recommended)
- **Quality:** High quality, clear and professional
- **Content:** Should represent the video content clearly
- **Alt Text:** Descriptive for accessibility

## How to Add Images:
1. Create or download appropriate thumbnail images
2. Name them exactly as listed above
3. Place them in this `frontend/public/thumbnails/` directory
4. The app will automatically use these images

## Placeholder URLs:
Currently using external YouTube thumbnails as placeholders. Replace with local images for better performance and reliability.
